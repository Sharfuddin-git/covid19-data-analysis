# 📊 COVID-19 Data Analysis Using Python

## 📌 Project Overview
This project analyzes COVID-19 data to understand trends, total confirmed cases, and comparison of death and recovery rates across different countries.

The analysis includes data visualization using bar charts and time-series plots.

---

## 🚀 Features

- Total confirmed cases per country
- COVID-19 case trends over time
- Death rate vs Recovery rate comparison (latest date)
- Data visualization using Matplotlib
- Basic data preprocessing using Pandas

---

## 🛠 Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib
- Jupyter Notebook

---

## 📂 Dataset

The dataset contains the following columns:

- Country
- Date
- Confirmed
- Deaths
- Recovered

---

## 📈 Key Insights

- Identifies the country with the highest confirmed cases
- Shows how cases evolved over time
- Compares death and recovery rates for the latest available date

---

## ▶ How to Run the Project

1. Install Python (version 3.8 or higher)
2. Install required libraries.
